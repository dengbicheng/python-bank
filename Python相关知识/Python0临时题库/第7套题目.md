## 任务2：
学校有近千名学生，在操场上排队，5 人一行余 2 人，7 人一行余3 人，3 人一行余 1 人，编写一个程序求该校的学生人数。
要求：使用分支、循环结构语句实现，直接输出结果不计分。

1.第一种方法
```python
#!/usr/bin/python 
# -*- coding: utf-8 -*-
n = 0
for x in range(1, 1000, 1):
    if x % 5 == 2 and x % 7 == 3 and x % 3 == 1:
        n = x
print(f"人数为{n}个")
```
