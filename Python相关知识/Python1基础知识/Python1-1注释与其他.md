# Python1-1注释与其他

## 1 Python注释

#### 1.1 四基础注释方法

1.第一种方法

```python
# 注释内容
```

2.第二种方法

```python
'''
注释内容
'''
```

3.第三种方法

```python
"""
注释内容
"""
```

4.第四种方法

```python
def a():
    '''这是文档字符串'''
    pass
print(a.__doc__)
#输出格式
'''
这是文档字符串
'''
```

注意：注释应该放在函数的第一行

## 2 其他一

#### 2.1 一行多个语句

```python
#举例格式
print("狗叫什么");print("没用的")   #使用；分隔开。

#输出格式
'''
狗叫什么
没用的
'''
```

#### 2.2 一个语句分成多行

```python
#举例
a = b = c = 10
x = a + b + c + 12
print(x)     #输出x
#方法一（使用“/”）
y = a +\    #
    b +\
    c +\
    12
print(y)
#方法二（使用括号）
z = (a +    #此处可以加上批注
     b +
     c +
     12)
print(z)

#输出格式（输出相同）
'''
42
42
'''
```
